# 이진 검색 (Binary Search)
= 자료구조 기반으로 정렬되어있는 데이터 안에서 특정 값을 찾는 기법

## 평균 시간 복잡도 : O(log n)

## 구현방법 및 메서드 
  반복문 : binarySearch_loop()
  재귀: binarySearch_recursive()

## 알고리즘 동작방식
  
  순차 검색 : 1 - 9 처례대로 for문으로 훑어서 원하는 값 찾는다.
  이진 검색 : 가운데부터 찾는다 -> 효율적
    => 5 < 6  --> 찾을 값이 더 크니 오른쪽 탐색
    => 7 < 6  --> 찾을 값이 더 작으니 왼쪽 탐색
    => 6 == 6  --> 값의 위치와 존재 여부 확인
