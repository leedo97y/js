1. HTTP
통신 프로토콜 : 컴퓨터나 원거리통신에서 메세지를 주고 받는 통신 규약 / 원칙
브라우저 안의 정보에 대한 어떠한 문서를 주고 받음 -> HTTP 통신 


2. Ajax 
: 비동기 자바스크림트와 XML의 약자 -> 요즘은 json를 쓴다.
-> 문서를 주고 받지 않고 바로 데이터를 가져올 수 있음 
(페이지 전환 없이 데이터를 가져옴)
등장 -- 페이지 전환에도 많은 영향을 끼침 / 웹 생태계가 변함
예) 무한 스크롤 


3. XMLHttpRequest
요즘은 json 데이터 포맷 많이 쓴다.


4. fetch API
: 부족한 부분을 보완, 강력하고, 유연함 
1) URL fetch 요청
2) fetch 응답 객체를 가져옴
3) 응답 객체가 JSON => 순수 JS
